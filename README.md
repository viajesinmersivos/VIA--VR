# VIA – Viajes Inmersivos Asistidos

Componentes publicables del proyecto VIA, desarrollados principalmente con:

- Unity (URP) + Meta XR / Quest SDK – Cliente VR para Meta Quest  
- Blender – Escenas y soportes para 180º/360º  
- DaVinci Resolve + Canon EOS VR Utility – Edición de vídeo VR180  
- Python (PyTorch, OpenCV) – Procesado y análisis de datos  
- Max / RNBO – Prototipado de audio interactivo  
- Backend web / API REST – Gestión de sesiones y métricas
