# VIA – Viajes Inmersivos Asistidos

Componentes publicables del proyecto VIA, desarrollados principalmente con:

- **Unity (URP) + Meta XR / Quest SDK** – Cliente VR para visores Meta Quest
- **Blender** – Escenas y soportes para contenido 180º/360º
- **DaVinci Resolve + Canon EOS VR Utility** – Edición de vídeo VR180 estereoscópico
- **Python** (PyTorch, OpenCV) – Procesado y análisis de datos y métricas de sesión
- **Max / RNBO** – Prototipado de audio interactivo y lógica audio-reactiva
- **Backend web / API REST** – Gestión de sesiones, dispositivos y almacenamiento de métricas
